CPHEditor#addAutocomplete
Create a custom autocompletion detector to add your own autocomplete box. The enableFn takes the form function (editor, selection, inString, inComment) and can return any sort of result you want. When rendered, if enableFn() returns a truthy value, the editor will dispatch an autocomplete event with the parameters (editor, name, result, cursorRectangle). You can use this to build your own autcompletion element.
stringname The name of the autocompletion handler
functionenableFn The autocompletion enablement handler
CPHEditor#addHotkey
Adds a hotkey handler with the format function (value, cursors) { ... } If a hotkey is added, the default behavior will be prevented.
stringkey Hotkey format in order ctrl+alt+shift+key
functionfn Hotkey handler
CPHEditor#animateNo
Shakes the editor back-and-forth, indicating no action can be taken.
CPHEditor#blur
Blurs the editor (removes focus)
CPHEditor#canSuggest
Tells us whether or not the editor can currently return typeahead or other suggestions.
CPHEditor#clearHistory
Clears all user action history. This *does not* re-render the editor.
CPHEditor#clearMetadata
Clears metadata on the editor, if needed. This will dispatch a metadata event and metadata/${key}` event that can be listener to via editor.on('metadata', fn).
stringkey The metadata key to clear
CPHEditor#disable
Disables the editor. The user can not make further changes or scroll the editor.
CPHEditor#enable
Enables the editor.
CPHEditor#find
Open the find and replace dialog for the editor.
stringvalue The value to search for

Finds the complement of a character at a specific index based on the language dictionary. e.g. finds } when { is located at the specified index.
stringvalue The value to search through
integern The character index to check
arrayResult in format [leftIndex, rightIndex]

Determines whether a specific character index is within a comment or not based on the language dictionary.

CPHEditor#inString
Determines whether a specific character index is within a string or not based on the language dictionary.

CPHEditor#open

CPHEditor#render
Renders the editor on the next available frame, if applicable. Multiple calls to .render will not negatively impact performance, extra calls are debounced.
stringvalue The value to render within the editor, should typically be editor.value or editor.getValue()
booleanforceRender Force an update on the next frame
CPHEditor#save
Dispatches a "save" event and creates a history entry if the user has performed an action. You can listen to this action via editor.on('save', function (editor, value) { ... })
stringvalue The value to set in the editor
string
CPHEditor#scrollPage
Scrolls the editor by a visible "page" amount based on the height of the editor. This *will* trigger a re-render.
stringdirection Either "up" or "down"
CPHEditor#scrollToText
Scrolls to the currently selected text in the editor. This *will* trigger a re-render.
CPHEditor#scrollToLine
Scrolls to a specific line index in the editor. This *will* trigger a re-render.
integerindex The line index to scroll to
CPHEditor#scrollTo
Scrolls the editor to a specific (x, y) coordinate from the top left of the editor. This *will* trigger a re-render.
integerx the x coordinate (from left)
integery the y coordinate (from top)
CPHEditor#select
Selects text in the editor at a specific start and end index. This *will* trigger a re-render.
integerstart The start index to select
integerend The end index to select

CPHEditor#setFormatter
Sets a custom formatter for a language. Custom formatter functions take the form function (line, inString, inComment) { ... } where inString and inComment are determined by the language dictionary.
stringlanguage The language the formatter applies to
functionfn The formatter function
CPHEditor#setLanguage
Sets the language for the editor. This *does not* automatically re-render the editor.

stringThe active language
CPHEditor#setMaximized
Set maximized mode on the editor so that it takes up all of its relative parent's height. This *does not* trigger a re-render. You must do that manually.
booleanvalue
CPHEditor#setMetadata
Sets metadata on the editor, if needed. This will dispatch a metadata event and metadata/${key} event that can be listener to via editor.on('metadata', fn).

stringvalue The metadata value to set

Set read-only mode on the editor. This *does not* trigger a re-render. You must do that manually.

stringvalue The value to set in the editor
string

stringname The name of the user action to dispatch
